# Portafolio - Modelo de página web para Agencia de Marketing

## Descripción
Este proyecto es mi primer modelo de página web, diseñado para una **agencia de marketing**. El objetivo es practicar y mostrar mis habilidades en **HTML y CSS**, creando una estructura funcional.

El proyecto utiliza una plantilla descargada de **[W3Layouts](https://w3layouts.com/)** bajo su **Starter License**. Según la licencia, se mantiene el crédito en el footer: *"Design by W3Layouts"*.

## Tecnologías utilizadas
- HTML5
- CSS3
- Imágenes libres de uso (de fuentes gratuitas como Unsplash, Pixabay o similares)
- Plantilla original de W3Layouts (con crédito en el footer)

## Características
- Diseño moderno y limpio.
- Menú de navegación funcional.
- Secciones: Inicio, Servicios, Sobre nosotros, Contacto.
- Responsive (adaptable a diferentes tamaños de pantalla).
